# TraxOsint

TraxOsint is a modular OSINT tool for IP intelligence using public APIs.

## Features
- IP Geolocation
- ISP / ASN lookup
- WHOIS intelligence
- Modular async architecture

## Requirements
- Python 3.11.x
- Windows / Linux

## Installation
pip install -r requirements.txt

## Usage
python traxosint.py
